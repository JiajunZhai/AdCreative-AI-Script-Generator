# 🥑 Avocado Workspace Hub (SOP 工业级生产引擎)

面向全球买量 (UA) 与广告创意团队的“交钥匙 (Turnkey)”智能辅助平台。
输入简单的游戏名、应用链接或创意角度，Avocado 将自动通过云端顶尖大模型矩阵，为您生成结构化、高转化率的分镜脚本 (SOP) 与多语言宣发文案。

## 🌟 核心价值与功能

- **🚀 极速智能生成 (SOP / Quick Copy)**
  抛弃繁杂的 Prompt 编写！只要输入核心受众与卖点，系统将自动生成规范化的“画面分镜 + 旁白口播 + 心理学洞察”三维工业级脚本。支持切换“极速文案”模式批量生成多语言广告语。
- **🛡️ 零偏离合规雷达 (Compliance Radar)**
  内置企业级词库拦截墙。在生成文案的瞬间，自动扫描并在仪表盘中高亮标注所有涉及“夸大、封号风险、黑话”的违禁词汇。支持一键“热重载 (Hot Reload)”，运营修改规则后秒级全网生效。
- **🤖 智算网络防断供 (Multi-hop Failover)**
  原生集成 DeepSeek, 阿里云百炼, 硅基流动等全球顶级大模型。当您的首选模型遭遇“网络抽风”或“限流报错”时，Avocado 将在 0.5 秒内自动切换至备用模型，保证产线永不停工。
- **📊 资产复用大盘 (Dashboard & @OUT)**
  所有生成的剧本与素材文案均可通过高密度控制台 (Dashboard) 进行二次比对 (Compare)、多语言转译 (Refresh)，并一键导出为标准的 Markdown/Excel 交接文档。

## 🚀 断网环境离线部署指南 (Air-gapped Deployment)

Avocado 已为您打包为**“纯净单体、一键点火”**的工业级离线部署包。无论您的局域网服务器是否连接外网，均可三步完成集群部署。

### 部署前序准备
请向您的研发获取最新生成的 `Avocado_Offline_Release.zip` 文件。该文件包含了 Avocado 运行所需的全部应用、数据和配置。

### 第一步：传输与解压
将 `Avocado_Offline_Release.zip` 拷贝至您的目标局域网服务器，并解压到一个专属目录中。

### 第二步：导入单体引擎
在解压出的文件夹中打开终端，执行以下指令将双端应用引擎装载进您的服务器：
```bash
docker load -i avocado-offline-images.tar
```

### 第三步：一键点火
引擎装载完毕后，执行起飞指令：
```bash
docker-compose up -d
```

🎉 **恭喜！** 此时系统已在后台静默运行。
您的团队现在可以打开浏览器，访问 `http://<您的服务器IP>:951` 登入 Avocado 工业级协作平台。

*(注：系统产生的所有知识库数据、合规规则与文案都会安全地保存在该目录的 `backend/data` 与 `@OUT` 文件夹中，容器重启或删除绝不会丢失您的任何资产。)*

## ⚙️ 系统初始配置

首次登入后，请按照以下步骤激活大模型引擎：

1. 点击左侧导航栏底部的 **"Providers" (云端引擎池)** 图标。
2. 在大模型控制台中，找到系统推荐的引擎（如 DeepSeek 或 阿里云百炼）。
3. 填入您的 `API Key`，点击保存。系统将自动验证网络连通性。
4. 返回 **Workspace Hub**，新建一个项目，开始您的自动化创意生产！

## ❓ 常见问题 (FAQ)

**Q: 为什么生成时提示 "CLOUD_UNAVAILABLE"？**
A: 请前往 Providers 面板，确认您至少配置并启用了 1 个可用的大模型 API Key，且左侧面板的网络连通性测试为绿色 (Pass)。

**Q: 发现有新的违禁词，如何加入拦截系统？**
A: 请在服务器后台打开 `backend/data/compliance/risk_terms.json` 文件进行编辑添加。保存后，在系统前端的“合规管控中心”点击右上角的 `HOT RELOAD (热重载)` 按钮，拦截机制即可瞬间对全网生效。

**Q: 如何获取我之前生成的所有 Word/Markdown 脚本文件？**
A: 除了在前端 UI 点击导出外，在您部署的 `Avocado_Offline_Release` 文件夹内，有一个名为 `@OUT` 的核心文件夹，所有生成的物理文档都会持久化自动备份在这里，可供随时打包取用。
